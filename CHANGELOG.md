- Container support (Docker, Kubernetes)
- Linting configuration (ESLint, Prettier, EditorConfig)
- Middleware system
- Code transformers
- Test utilities

### Features
- `veritas generate <file>` - Generate tests from source
- `veritas record <url>` - Record API traffic
- `veritas mock <file>` - Generate mocks from traffic
- `veritas auto <file>` - Full workflow (record + generate)
- `veritas analyze <file>` - Analyze code structure
- `veritas report <file>` - Generate test reports
- `veritas cache` - Cache management
