# Veritas Examples

This directory contains examples for using Veritas.

## Running Examples

```bash
# Install dependencies
npm install

# Run basic example
npx tsx examples/basic.ts

# Run traffic recording
npx tsx examples/record.ts http://localhost:3000

# Run auto generate
npx tsx examples/auto.ts src/components/Button.tsx http://localhost:3000
```

## Examples

### basic.ts

Basic example of analyzing code and generating tests.

```bash
npx tsx examples/basic.ts
```

### record.ts

Record API traffic and generate mocks.

```bash
npx tsx examples/record.ts http://localhost:3000
```

### auto.ts

Full workflow: record traffic, analyze code, generate tests, and run tests.

```bash
npx tsx examples/auto.ts src/components/Button.tsx http://localhost:3000
```

### plugins.ts

Example of creating custom plugins.

```bash
npx tsx examples/plugins.ts
```
